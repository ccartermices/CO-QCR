# CO-QCR: Causal Order Preserved Quantum Combinatorial Reasoning

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)

## Overview

CO-QCR (Causal Order preserved Quantum Combinatorial Reasoning) is a quantum-inspired method for selecting causally consistent chain-of-thought reasoning paths from multiple candidate responses generated by large language models. This repository contains the implementation of the CO-QCR algorithm as described in our paper.

Chain-of-thought reasoning has emerged as a powerful paradigm for enhancing the reasoning capabilities of large language models. However, existing methods for selecting optimal reasoning paths often neglect the causal dependencies between reasoning steps, leading to logically inconsistent conclusions. CO-QCR addresses this limitation by incorporating explicit causal constraints into the quantum optimization process.

## Key Features

- **Causal Consistency Guarantee**: Achieves perfect causal consistency in reasoning path selection
- **Quantum Optimization**: Uses QAOA (Quantum Approximate Optimization Algorithm) for efficient combinatorial optimization
- **Multi-Model Support**: Compatible with various LLM APIs (OpenRouter, Zhipu AI)
- **Comprehensive Evaluation**: Includes coherence and causal consistency metrics

## Usage

### Step 1: Download Dataset

Download and sample questions from the Chinese-SimpleQA benchmark:

```bash
python step1_download_dataset.py
```

This script downloads the Chinese-SimpleQA dataset and randomly samples 20 questions for evaluation.

### Step 2: Collect Chain-of-Thought Responses

Collect CoT responses from language models:

```bash
python step2_api_call.py
```

**Configuration:**
- Create a `key.txt` file containing your OpenRouter API key
- Create a `zhipukey.txt` file containing your Zhipu AI API key
- The script will collect 50 responses per model (GLM-4.5-Air, Minimax-M2.5, GLM-4.7-Flash)

### Step 3: Run Quantum Optimization

Run the QCR-LLM vs CO-QCR comparison:

```bash
python step4_quantum_test.py
```

This script:
1. Extracts reasoning fragments from collected responses
2. Identifies causal pairs between fragments
3. Constructs Hamiltonians for both methods
4. Runs variational quantum optimization
5. Computes coherence and causal consistency metrics

### Step 4: Visualize Results

Generate comparison charts and tables:

```bash
python step5_visualize.py
```

### Run All Steps

Use the main control script to run all steps sequentially:

```bash
python main.py
```

The main script automatically tracks progress and skips completed steps.

## Method

### Problem Formulation

Given a question and multiple candidate chain-of-thought responses, CO-QCR selects a subset of reasoning fragments that form a coherent and causally consistent reasoning path. The selection is formulated as a binary optimization problem where each fragment is associated with a binary variable indicating its selection.

### Hamiltonian Construction

The optimization objective is encoded as a Hamiltonian with three terms:

1. **Quality Term (H_Q)**: Encourages selection of relevant and reliable fragments
2. **Coherence Term (H_C)**: Encourages selection of semantically related fragments
3. **Causal Term (H_CA)**: Penalizes selections that violate causal dependencies

The total Hamiltonian is:
```
H = αH_Q + βH_C + H_CA
```

### Causal Pair Identification

Causal pairs are identified through:
- Explicit causal markers (therefore, because, thus, etc.)
- Sequential dependency analysis between adjacent fragments
- Semantic overlap detection between premises and conclusions

### Variational Quantum Optimization

The optimization uses QAOA with:
- 8 qubits for fragment representation
- 2 QAOA layers
- Adam optimizer for parameter tuning
- Parameter-shift rule for gradient computation

## Results

| Metric | QCR-LLM | CO-QCR | Change |
|--------|---------|--------|--------|
| Average Coherence | 0.327 | 0.315 | -3.9% |
| Average Causal Consistency | 0.821 | 1.000 | +21.7% |
| Questions Processed | 14 | 14 | - |

**Key Findings:**
- CO-QCR achieves perfect causal consistency across all test cases
- The improvement comes at a modest 3.9% decrease in coherence scores
- The trade-off between coherence and causal consistency is fundamental to reasoning selection

## Extending CO-QCR

### Adding New Models

To add support for additional language models, modify `step2_api_call.py`:

```python
OPENROUTER_MODELS = [
    'z-ai/glm-4.5-air:free',
    'minimax/minimax-m2.5:free',
    # Add your model here
]
```

### Custom Causal Detection

To implement custom causal pair detection, modify the `identify_causal_pairs` function in `step4_quantum_test.py`.

### Different Optimization Parameters

Adjust the quantum optimizer settings:

```python
qcr_solver = QCR_LLM_Solver(
    n_qubits=8,      # Number of qubits
    n_layers=2,      # QAOA layers
    lr=0.01,         # Learning rate
    n_iterations=100 # Optimization iterations
)
```
